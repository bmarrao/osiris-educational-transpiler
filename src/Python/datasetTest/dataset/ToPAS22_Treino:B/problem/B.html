Problema B: Quantas rifas já vendi?


Para angariarem uns cobres para a viagem de finalistas, a turma da Mariazinha decidiu rifar um cabaz de bens alimentares (doados pelos pais). A Mariazinha começou logo a vender rifas, anotando num caderno o nome e a quantidade de rifas que vendeu a cada pessoa. O problema é que, sempre que lhe perguntam quantas rifas já vendeu, tem de somar um enorme número de parcelas. Como seria bom ter um programa que fizesse as contas...

Tarefa

Escreva um programa que, dados o nome e a quantidade de rifas que a Mariazinha vendeu a cada pessoa, indique quantas rifas ela já vendeu.

Input

A primeira linha do input tem um inteiro, \(P\), que representa o número de pessoas a quem a Mariazinha vendeu rifas. Por cada pessoa, é fornecida a seguinte informação:
  • A primeira linha relativa a uma pessoa contém o nome da pessoa, que é formado por (de 1 a 25) letras maiúsculas ou minúsculas, sem acentos nem cedilhas.
  • A segunda linha tem um inteiro, \(R\), que é o número de rifas vendidas a essa pessoa.

Restrições

  \(1 \leq P \leq 100\) Número de pessoas
  \(1 \leq R \leq 50\) Número de rifas vendidas a uma pessoa

Output

Uma única linha com um inteiro que indica o número total de rifas vendidas pela Mariazinha.

Exemplo

Input 

3
Fulano
2
Beltrano
18
Sicrano
10

Output 

30
