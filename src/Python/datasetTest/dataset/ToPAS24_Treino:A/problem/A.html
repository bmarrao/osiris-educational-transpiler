Problema A: Ambiente-se!


Não é de problemas ambientais que vamos falar, dos que ameaçam a vida, como os 60ºC de sensação térmica sentidos no Rio de Janeiro, a 16 de março de 2024. Vamos apenas classificar inteiros para testar outros ambientes.

Tarefa

Escreva um programa que, dado um inteiro, indica se esse número é zero, positivo ou negativo.

Input

O input tem uma linha com um inteiro \(N\).

Restrições

  \(-100 \leq N \leq 100\) O número inteiro

Output

O output tem uma única linha com o inteiro \(N\), seguido de um espaço, e de "zero!", "positivo!", ou "negativo!", conforme \(N\) seja zero, positivo ou negativo.

Exemplo 1

Input 

-11

Output 

-11 negativo!

Exemplo 2

Input 

12

Output 

12 positivo!

Exemplo 3

Input 

0

Output 

0 zero!
