Problema A: É quadrado?


Um quadrado é um retângulo cujos lados têm todos o mesmo comprimento.

Tarefa

Escreva um programa que, dados os comprimentos de dois lados adjacentes de um retângulo, calcule a sua área e detete se o retângulo é um quadrado.

Input

O input tem uma única linha com dois inteiros, \(C_1\) e \(C_2\), que representam os comprimentos (em metros) de dois lados adjacentes do retângulo.

Restrições

  \(1 \leq C \leq 1\,000\) Comprimento de um lado

Output

A primeira linha do output tem a área do retângulo (em metros quadrados). Se o retângulo for um quadrado, e apenas nesse caso, há uma segunda linha com a palavra QUADRADO (em maiúsculas).

Exemplo 1

Input 

5 5

Output 

25
QUADRADO

Exemplo 2

Input 

10 5

Output 

50
